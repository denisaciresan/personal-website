MS SQL SERVER

Creare interogări, proceduri și funcții

Data realizării: Noiembrie - Decembrie 2023

Proiectul realizat utilizează Microsoft SQL Server pentru a gestiona o bază de date complexă dedicată unui magazin de produse cosmetice. Datele din această bază de date sunt fictive și au fost create pentru a simula funcționalitățile și operațiunile tipice ale unui astfel de magazin. Proiectul include crearea de tabele, popularea acestora cu date fictive și implementarea unor proceduri și funcții pentru a gestiona și afișa informațiile stocate.

Baza de date este organizată în mai multe tabele interconectate, fiecare având un rol specific:

  • ClientiCosmetice: Stochează informațiile despre clienți.
  • ComenziCosmetice: Stochează informațiile despre comenzile plasate de clienți.
  • DetaliiComandaCosmetice: Conține detalii specifice fiecărei comenzi, cum ar fi produsele comandate, cantitatea și prețul.
  • ProduseCosmetice: Stochează informațiile despre produsele disponibile în magazin.
  • CategoriiCosmetice: Stochează categoriile de produse cosmetice.

Această interogare calculează totalul încasărilor pentru fiecare categorie de produse livrate, oferind o imagine de ansamblu asupra performanței financiare a magazinului în funcție de categoriile de produse.

Această funcție returnează stocul rămas al unui produs specific după finalizarea vânzărilor și al produselor aflate în procesare. Este esențială pentru gestionarea inventarului și pentru a evita epuizarea stocurilor.

Această procedură permite afișarea comenzilor în funcție de statusul acestora, data comenzii, produsele comandate, cantitatea, prețul și valoarea totală a comenzii. Este utilă pentru a monitoriza comenzile în diverse stadii, cum ar fi "In procesare", "Finalizat" sau "Livrat".
